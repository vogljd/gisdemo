• Hover your mouse over the various buttons/tools to get a description of their functionality.
  • When zoomed in all the way, click the map to see parcel information.
  • If you don't see the desired imagery layer, make sure the other layer transparencies are not set to 100% on.
  • Open and close the on-sceen tool drawers by clicking the handles at the edge of the tool drawers.
  • The Larimer County Enterprise GIS web mapping index page can be accessed by right-clicking the map and choosing the "Open LC Maps Index Page" option.

    This will open a new web browser window at the current location. Click the maps on that page to access all the Larimer County online mapping products.
  • The overview map is interactive. You can drag/drop the map extent rectangle to repostion the main map.
